# ✈️ Travel & Rekomendasi Destinasi menggunakan Metode Sistem Penunjang Keputusan (SPK SMART) ✈️

## ✨ Fitur Utama dari website ini ✨

* **SPK SMART :** Menghitung dan Merangking Algoritma paket wisata terbaik secara *real time* berdasarkan Benefit dan cost
* **Dashboard Analitik :** Visualisasi data pendapatan bulanan dan destinasi terlaris
* **Manajemen Paket Wisata & Pesanan :** Fitur CRUD untuk mengelola data paket liburan
* **Pelanggan :** Halaman profil khusus pelanggan untuk memantau pesanan 
* **Sistem Autentikasi & Role :** Pembatasan hak akses Admin, Staff dan Costumer

## 🛠️ Tech Stack 🛠️

* **Framework :** Laravel 12
* **Frontend :** Bootstrap, AdminLTE 3
* **Database :** MySQL
